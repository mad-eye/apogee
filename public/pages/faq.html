Installation Issues

Can I install MadEye on Windows?
Probably. It should work just fine on Windows 7+, but it hasn't been thoroughly tested and is not officially supported. You don't need to use `sudo` for `npm install`, of course.
Can I install MadEye on OS X?
We’ve tested it thoroughly on 10.8. It should work on any installation the latest version of NodeJS works on.
Can I install MadEye on Linux?
We use it frequently on Ubuntu, but it should work on any standard Linux distribution that NodeJS supports. Note that the default nodejs package on Ubuntu 12.04 is quite out of date (0.6.x); you should consider installing it with the executable, or via Chris Lea's apt repo.
Can I install NodeJs using Homebrew?
Yes! You can install node via Homebrew, but sometimes Homebrew doesn't include the npm modules on its path. Make sure that you include /usr/local/share/npm/bin in your PATH.
What's the minimum version of NodeJS required for MadEye?
v0.8.21
How come I get npm ERR! Error: No compatible version found: when I try to install MadEye?
You have an outdated version of NodeJS. Install the latest executable.
It says it installed MadEye, but when I type `madeye`, it says “command not found”.
Check to make sure that /usr/local/share/npm/bin is in your PATH.
The online project is acting a little funny, but I haven't saved the file yet!
No worries, the unsaved contents of the file are stored behind the scenes. Just refresh, and you should be fine.
