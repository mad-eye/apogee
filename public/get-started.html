Getting Started

Quick Start

  1. Verify that you have node.js (v0.8.21 or greater) installed. 
  2. Type sudo npm install -g madeye in your terminal.  MadEye's installed!
  3. Type  madeye  in the directory to share

Now your project is up and running! Send the URL to your teammates and start editing.

Check out the FAQ for more information.

 

How does it work?

MadEye watches your project on your file system, keeping it in sync with the cloud editor. It keeps the changes you and your friends make in sync with each other as well, so everyone can see what each other is doing. When you're ready, you can save the changes back to your file system!
